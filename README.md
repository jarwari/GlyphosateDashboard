# Prenatal Glyphosate, AMPA, and Oxidative Stress Dashboard

This dashboard explores associations between prenatal glyphosate and AMPA exposure and oxidative stress biomarkers using data from the Illinois Kids Development Study (I-KIDS).

## Features
- Interactive forest plots of adjusted linear regressions (beta coefficients and 95% confidence intervals)
- Correlation heatmap with hover-based interpretation
- Filtering by exposure and biomarker

## Data
The dataset used in this dashboard is confidential and cannot be shared publicly. The analysis is based on a subset of the I-KIDS prospective birth cohort (2012–2017).

## Public Health Importance
Glyphosate is widely used, yet human evidence on prenatal glyphosate and AMPA exposure in relation to oxidative stress during pregnancy remains limited. This dashboard helps communicate whether exposure patterns are associated with oxidative stress biomarkers and supports continued investigation into potential biological effects during a sensitive developmental window.

## How to Run
1. Open `final-dashboard.Rmd` in RStudio  
2. Click **Run Document**
